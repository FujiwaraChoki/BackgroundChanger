# BackgroundChanger Python
This simple little script changes the background image on most Linux Distros.
Tested on Ubuntu 20.04/22.04, Fedora, Garuda (Mate) and Kali.

# Prerequisites
1. PySimpleGUI
```
pip3 install PySimpleGUI
```
